MUSIQUEUE 2.0

is an application created by Noam Zilberstein, Lucas Peña, Tayler Mandel, and Tiernan Garsys. It is an easy to use app made to help hosts throw the perfect party.

HOW IT WORKS

MUSIQUEUE 2.0 allows event hosts to construct a playlist for their event. Our algorithm uses the Facebook Graph API to determine the musical preferences of the event attendees and uses this information to prioritize artists in the playlist. Artists that are 'liked' by more attendees are more likely to appear in the play queue. This algorithm creates a unique, diverse playlist that is guaranteed to satisfy all guests and can be played in app via Spotify.
